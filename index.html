Como funciona

  • Fhellype = Sanji
  • Clara = Nico Robin
  • Modo GIF = busca real na API do GIPHY
  • Modo imagem = imagens locais prontas no site

API key

Para o modo GIF funcionar, abra o arquivo script.js e coloque sua chave do GIPHY em const GIPHY_API_KEY.

Google Imagens

O Google Imagens não oferece uma integração front-end simples para esse caso. Por isso, deixei a parte de imagens funcionando com galeria local, e a parte de GIF com API real do GIPHY.
