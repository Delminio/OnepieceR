COMO USAR

1. Abra o arquivo script.js
2. Encontre esta linha:
   const GIPHY_API_KEY = 'COLE_SUA_CHAVE_AQUI';
3. Substitua pela sua chave real do GIPHY
4. Abra o index.html no navegador

OBSERVACOES IMPORTANTES

- O modo GIF usa a API real do GIPHY.
- O modo imagem usa uma galeria local do proprio site.
- Google Imagens nao foi integrado diretamente porque isso nao funciona de forma confiavel apenas no front-end.
- Se quiser, depois eu posso adaptar essa estrutura para funcionar com back-end, banco de dados ou upload de imagens proprias.
